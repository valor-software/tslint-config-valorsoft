## Philosophy
- contains all rules explicitely
- almost all rules enabled

## Install
```sh
npm install tslint-config-valorsoft
```

```js
// tslint.json
{
  "extends": "tslint-config-valorsoft",
  "rulesDirectory": "./node_modules/codelyzer/dist/src"

  "rules": {
    // your customization
  }
}
```
